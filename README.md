# steel-chair

## What is this?

The primary goal of this project is learning and fun, as is the case for all of
my personal projects.

My development plan is to build iteratively, adding features one-at-a-time.
I have no idea of what "complete" looks like, and I do not intend to produce a
"final" version of this project. The starting idea was the most bare-bones
version possible of a pro wrestling match simulator.

The result of this development plan is that I can't "plan ahead" the way I
normally would. The code and architecture will evolve organically, step-by-step.
This is a very different experience, because all of my other projects have had
an "end goal" to work toward from the start. Whereas with this project, I'm
interested in seeing how the code changes each step along the way.
